Datasets for the Key Point Generation Task:


The IBM Debater data sets ArgKP-2021 (27.5K) and ArgQ-Rank-30kArgs used for the fine-tuning of the argument quality and key point summarization task are available at https://research.ibm.com/haifa/dept/vst/debating_data.shtml.

For reasons of privacy, the Twitter dataset may not be published in its original form. Therefore, only the tweet IDs of the dataset are available.
To reproduce the data collection process using the Twitter API, both the python script and the list of MP IDs used to collect the Twitter dataset are provided.
